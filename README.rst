
=======================
tutorial
=======================

This is a Django REST framework sample project.


Requirement
===========

:Python: 3.7.0
:Django: 2.2.4
:djangorestframework: 3.10.2
:pytz: 2019.2

Quick start
===========
1. clone project::

    git clone https://github.com/kzmrt/tutorial.git

2. start server::

    python manage.py runserver
