
=======================
tutorial

This is a Django REST framework sample project.


Requirement
===========

:Python: 3.7.0
:Django: 2.1.7
:djangorestframework: 3.9.2
:pytz: 2018.9

Quick start
===========
1. clone project::

    git clone https://github.com/kzmrt/tutorial.git

2. start server::

    python manage.py runserver
